Help & usage guide

Simple instructions for staff members using the system.

How to use

1) Login
Use your staff username and password to enter the system.

2) Add Reservation
Enter a unique reservation number, guest details, room type, and dates. Then save.

3) View Reservation
Use reservation number to retrieve full booking details.

4) Billing
Enter reservation number to calculate number of nights and total amount automatically.

Common issues
• Duplicate reservation number will be rejected.
• Check-out date must be after check-in date.
• Contact number must be a valid numeric format (with optional +).
If you face errors, contact the system administrator.
